== Description ==

Displays the current infocom status, as announced by ISC (http://isc.sans.org)
in a widget. You can choose between, text, colored text, small and regular image to display.

== Installation ==

1. Unzip and upload the complete directory `isc-alert-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget in your widget
